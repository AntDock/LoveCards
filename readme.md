# LoveCards2  

### DEV环境  
PHP8+Composer+mysql  

### TP6依赖安装  
composer install

### 基本配置 
关闭放跨站(宝塔)  
设置运行目录为Public  
导入数据库  
修改“em.env”文件为“env”并修改数据库信息
